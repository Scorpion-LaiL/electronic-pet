# AI Coding Pet 2.0 更新说明

## 版本信息

- 版本号：`2.0.0`
- 版本主题：`优化 UI、增加动物种类、完善照顾规则`

## 本次更新内容

### 1. UI 视觉升级

- 将主界面调整为更接近参考图的像素宠物风格。
- 重做宠物选择卡片、主舞台、气泡和桌面宠物展示区域。
- 新增从参考图提取的宠物贴图资源，整体视觉更统一。

### 2. 动物种类扩充

原有宠物种类：

- 小猫
- 小狗

新增宠物种类：

- 小猪
- 小狐狸
- 小乌龟

当前共支持 5 种宠物。

### 3. 宠物行为与成长表现优化

- 不同物种拥有不同的待机动作和互动节奏。
- 宠物成长阶段会通过体型和表现差异体现出来。
- 宠物在桌面场景中的存在感和辨识度更强。

### 4. 规则机制优化

- 喂食、清洁、玩耍、睡觉等照顾动作增加连续操作限制。
- 同类动作最多连续执行 2 次，之后需要等待 30 分钟。
- 状态衰减节奏重新平衡，更适合桌面陪伴式体验。

## 适用说明

`2.0` 版本重点从“首版可运行原型”升级为“更接近桌面宠物体验的可玩版本”，后续可继续接入真正的桌面壳能力，例如透明窗口、托盘菜单和桌面悬浮运行。
